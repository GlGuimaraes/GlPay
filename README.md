# GlPay

Um projeto que desenvolvi utilizando a IDE #AndroidStudio: a Login-Page da GLPAY. Uma empresa ficticia que inventei

Com ela os usuários têm a opção de entrar em suas contas existentes usando seu endereço de e-mail e senha. O design limpo e moderno do formulário de login facilita a inserção das informações necessárias, enquanto o botão de login é proeminente e convidativo.

Para novos usuários, oferecemos a conveniência de se cadastrar na hora. Com apenas alguns campos obrigatórios, como nome, e-mail e senha, o processo de registro é rápido e intuitivo.

Entendo que os esquecimentos de senha acontecem. Por isso, implementei um recurso de recuperação de senha para ajudar os usuários a redefinir suas senhas de forma segura. Ao clicar em "Esqueci minha senha", os usuários receberão instruções claras sobre

À medida que continuo a desenvolver o projeto , estou comprometido em receber feedback dos usuários para aprimorar constantemente a experiência. Estamos ansiosos para expandir os recursos da Login-Page!
